# Title
This project is intended to Landing Page for Cloud Stones.

# clone repo
```
git clone https://github.com/cloudstonesorg/csp-ui && cd csp-ui
```
# Install node packages
```
npm install
or
npm install --force
```
# Run in-build server
```
ng serve
```
# Testing
http://localhost:4200/
